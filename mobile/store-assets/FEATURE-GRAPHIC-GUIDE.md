# 피처 그래픽 (Feature Graphic) 가이드

## 사양
- **크기**: 1024 x 500 픽셀
- **형식**: PNG 또는 JPEG (PNG 권장)
- **용도**: Play Store 앱 상세 페이지 상단 배너

## 디자인 권장사항

### 1. 간단한 디자인 (권장)
```
┌──────────────────────────────────────────────────────┐
│                                                      │
│         [앱 아이콘]    매일일독                       │
│                        매일 성경을 읽는 습관           │
│                                                      │
└──────────────────────────────────────────────────────┘
```

- 배경색: `#EFECE8` (앱 테마 색상)
- 앱 아이콘을 왼쪽에 배치
- 앱 이름과 태그라인을 오른쪽에 배치

### 2. Canva 사용 (무료)
1. https://www.canva.com 접속
2. "사용자 정의 크기" → 1024 x 500 입력
3. 배경색 `#EFECE8` 설정
4. 앱 아이콘 업로드 및 배치
5. 텍스트 추가: "매일일독" + "매일 성경을 읽는 습관"
6. PNG로 다운로드

### 3. Figma 사용 (무료)
1. https://www.figma.com 접속
2. 1024 x 500 프레임 생성
3. 디자인 작업 후 Export

## 제공된 자산
- 앱 아이콘: `store-assets/icon-512.png`
- 앱 테마 색상: `#EFECE8`
- 앱 이름: 매일일독
- 태그라인: 매일 성경을 읽는 습관

## 주의사항
- 텍스트가 너무 작으면 안 됨 (최소 24pt 이상)
- 중요 콘텐츠는 중앙에 배치 (상하좌우 여백 확보)
- 배너가 다양한 기기에서 잘려 보일 수 있으므로 가장자리에 중요 요소 배치 금지

## 빠른 제작 옵션

피처 그래픽 없이도 Play Store에 앱을 등록할 수 있습니다.
하지만 피처 그래픽이 있으면 앱이 더 전문적으로 보입니다.

### 임시 솔루션
Play Console에서 "피처 그래픽"은 필수가 아닙니다.
나중에 추가해도 됩니다.
