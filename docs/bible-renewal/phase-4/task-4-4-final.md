## 테스트 시나리오

### 시나리오 1: 신규 사용자 플로우

```
1. 비로그인 상태로 /bible 접속
2. 창세기 1장 표시 확인
3. 다른 장으로 이동 (마태복음 5장)
4. 로그인 유도 확인 (북마크, 노트 등 클릭 시)
5. 로그인 후 기능 동작 확인
```

### 시나리오 2: 기존 사용자 - 읽기모드

```
1. 로그인 상태로 /bible 접속
2. 마지막 읽던 위치로 이동 확인
3. 성경 읽기 (스크롤, 장 이동)
4. "읽음으로 표시" 버튼 클릭
5. 토스트 메시지 및 상태 변경 확인
6. /bible/history에서 기록 확인
```

### 시나리오 3: 기존 사용자 - 통독모드

```
1. /plan 접속
2. 오늘 일정 클릭
3. 확인 모달 표시 → "읽기 시작"
4. /bible?plan=x&schedule=y 이동
5. 통독모드 인디케이터 확인
6. 마지막 장에서 다음 클릭
7. 완료 확인 모달 → "완료 처리"
8. 토스트 및 진도 업데이트 확인
```

### 시나리오 4: 부가 기능 전체

```
1. 북마크 추가 (장 단위)
2. 북마크 추가 (절 단위)
3. /bible/bookmarks에서 목록 확인
4. 북마크 클릭 → 해당 위치 이동

5. 묵상노트 작성 (빠른 메모)
6. 묵상노트 상세 편집
7. /bible/notes에서 목록 확인

8. 하이라이트 추가
9. 색상 변경
10. /bible/highlights에서 목록 확인
```

### 시나리오 5: 설정 테스트

```
1. /bible/settings 접속
2. 폰트 크기 변경 → 미리보기 확인
3. 테마 변경 → 전체 앱 적용 확인
4. 기본 진입점 변경
5. /bible 재접속 → 변경된 진입점 확인
6. 페이지 새로고침 → 설정 유지 확인
```
